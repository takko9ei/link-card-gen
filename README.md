**Link Card Generator** is a powerful and flexible portfolio builder designed to create personal portfolios with ease. Built with Next.js and Tailwind CSS, it features a highly interactive drag-and-drop interface.

### âœ¨ Features

- **Drag & Drop Layout**: Effortlessly arrange your cards and blocks. Move items between columns or reorder them within columns.
- **Multi-Column Support**: dynamic grid layouts supporting 1 to 3 columns to fit your content needs.
- **Rich Block Types**:
  - **Header Block**: Show off your profile with an avatar, name, interactive tags, and bio.
  - **Text Block**: Add descriptive text for about sections or notes.
  - **Gallery Block**: Display your work or photos in a grid.
- **Customization**:
  - Adjust column and block gaps.
  - Set custom background colors or upload background images.
  - Control block opacity.
  - Resize blocks vertically to fit content.
